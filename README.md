# GitHub AI Agent

AI-powered web agent that auto-analyzes code, generates READMEs, and deploys repos to GitHub with one click. Built with pure JS and GitHub API.

## Features
- Auto-Analysis: Instantly detects language, classes, functions, and imports.
- - AI Review: Real-time code debugging and best-practice suggestions.
  - - Smart README: Generates professional documentation automatically.
    - - One-Click Deploy: Creates and pushes files to GitHub without CLI.
     
      - ## Getting Started
      - 1. Open index.html in your browser.
        2. 2. Enter your GitHub Token (repo scope).
           3. 3. Paste your code and watch the magic happen!

              5. - Vanilla JavaScript
                 - - GitHub REST API
